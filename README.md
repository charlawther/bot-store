# Telegram Bot для Интернет-Магазина

Этот проект представляет собой Telegram-бота для интернет-магазина, который позволяет пользователям просматривать товары, добавлять их в корзину и оформлять заказы.

## Описание

Бот предоставляет следующие функции:
- **/start**: Приветствие и краткое описание функциональности бота.
- **/catalog**: Просмотр доступных товаров в магазине.
- **/cart**: Просмотр содержимого корзины с указанием общей суммы.
- **/checkout**: Оформление заказа и очистка корзины.
